# Retail Sales Data Analysis end-to-end
This project provides an end-to-end analysis of retail sales data using SQL, Python, Machine Learning, and interactive dashboards.


## Project Overview
This project provides an end-to-end analysis of retail sales data using
SQL, Python, Machine Learning, and interactive dashboards.

## Tools & Technologies
- Python (pandas, numpy, matplotlib, seaborn, scikit-learn)
- SQL (MySQL / PostgreSQL)
- Power BI / Tableau
- Streamlit
- Git & GitHub

## Key Analysis
- Sales trends analysis
- Customer segmentation
- Product performance
- Time-series forecasting
- Business KPIs
